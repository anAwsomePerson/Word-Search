# Word-Search
It generates word search puzzles. It appears to usually work.

Type words in in.txt and separate the words with line breaks. Run the thing, and then read out.txt. For example, 

    silky
	poison
	bone
	mix

outputs 

    qcymixw
    snbxssp
    bhnjmij
    oyiceln
    ncizckj
    erwaiym
    poisonh
