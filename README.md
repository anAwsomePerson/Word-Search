# Word-Search
It generates word search puzzles. It appears to usually work.

Type words in words.txt and separate the words with line breaks. Run Word-Search.jar, and then read answer.txt and puzzle.txt. For example, 

  silky
  poison
  bone
  mix

outputs 

  pb     
   o   m 
   ni  i 
   e s x 
      o  
       n 
  silky 
  
and

  pbbquqa
  tohsimq
  wnizfia
  jeesnxn
  niowods
  jiflknc
  silkysx
